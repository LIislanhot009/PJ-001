# Product Workflow Demo

一个脱敏的个人作品集静态演示页，用三类角色展示产品工作流。页面第一屏就是可点击工作台，并包含动效流程图、功能说明和模拟运行日志。

- 供应商入口：提交产品资料、查看公开风险摘要、生成改款建议
- 内部运营入口：选品分析、外观风险预检、部件级复核、改款复检
- 管理员入口：任务队列、运行健康、权限与审计

## 说明

本项目是纯前端 mock demo，可以部署到 GitHub Pages。页面不连接真实后台，不包含真实源代码、真实环境配置、真实客户数据或未脱敏业务数据。

## 本地预览

直接打开 `index.html`，或使用任意静态服务器：

```bash
python -m http.server 8080
```
